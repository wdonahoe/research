Use this script to apply any R script that takes a set of text files to some directory containing text files.

The data can be in the same folder as r_analyze.py or in a subdirectory of the folder containing
r_analyze.py.

Usage: 

1) ./r_analyze.py <R script> <folder> To analyze text files in folder.
2) ./r_analyze.py <R script> To analyze text files in this folder.

<R script> can be only the filename without extension, or with .R extension.
